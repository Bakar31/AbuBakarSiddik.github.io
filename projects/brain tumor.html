Brain Tumor Classification

  • Tech Stack: Python, Deep learning, Transfer Learning, CNN, EfficentNet
  • Github URL: Project Link

Goal: To minimize the number of surgeries and refine the type of therapy required

Motivation: Currently, genetic analysis of cancer requires surgery to extract a tissue sample. Then it can take several weeks to determine the genetic characterization of the tumor. Depending upon the results and type of initial therapy chosen, a subsequent surgery may be necessary. If an accurate method to predict the genetics of the cancer through imaging (i.e., radiogenomics) alone could be developed, this would potentially minimize the number of surgeries and refine the type of therapy required

My Approch:

There were to total 400116 files of Size 136.85 GB and Type dcm was given. A dedicated DataFrameIterator was constructed that reads the data dcm file as an array with a size of 224x224. The path associated with each file was saved in a dataframe, which was later utilized to load the file. I implemented EfficentNetB0-B3. Each architecture was trained for each type of MRI data. There were four types of scans.

  • Fluid Attenuated Inversion Recovery (FLAIR)
  • T1-weighted pre-contrast (T1w)
  • T1-weighted post-contrast (T1Gd)
  • T2-weighted (T2)

    • Total 12 models was trained. Models for each type of scan were then ensambled.
